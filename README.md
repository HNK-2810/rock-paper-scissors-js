# Rock Paper Scissors Game

A simple Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## Features
- User can choose Rock, Paper, or Scissors
- Computer generates a random move
- Displays the result (Win, Lose, or Draw)
- Shows explanation of the winning move (example: Paper beats Rock)
- Keeps track of the user and computer scores

## Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation & Events)

## How to Play
1. Click on Rock, Paper, or Scissors.
2. The computer will randomly select its move.
3. The game will display the result and update the score.

## Project Purpose
This project was built to practice JavaScript concepts like DOM manipulation, event handling, and basic game logic.
